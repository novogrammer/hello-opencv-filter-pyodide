# Hello OpenCV Filter Pyodide

demo https://novogrammer.github.io/hello-opencv-filter-pyodide/

## 前提
Node.js 20.x

## 環境構築

```bash
npm install
```

## ビルド

```bash
npm run build
```

## 開発

pyodideのパッケージをダウンロードする必要がある。
ダウンロードはビルドタスクに含まれている。
```bash
npm run dev
```

## プレビュー

ビルドが必要
```bash
npm run preview
```
